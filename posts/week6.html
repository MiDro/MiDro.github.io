Week 6

Monday June 26, 2017

Today, we were able to communicate with the Encompass board uploading a sketch to turn on GPIO pins. We used Keil to upload the relevant pieces, so we have confirmed that we receive some output from the circuit.

Tuesday June 27, 2017

We got the router set up and got DNS set up. However, a problem that we ran into was that DNS didn't work on all computers. Circuit design and PCB layout were also updated. The software team also worked on IPSO objects in order to send data to the CMS.

Wednesday June 28, 2017

The PCB layout had a lot of bugs, so we continued with alterating of the PCB today. We were definitely not done yet. We also set up DNS and tried to get the gateway set up.

Thursday June 29 2017

The design of the first iteration of the PCB is complete. We also finished setup of the Flask API to the PI Server.
